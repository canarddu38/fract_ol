# fract_ol
Computer Graphics Fractals

## Command line options
`./fract_ol type set`

Where `type` is 
- `m` for Mandelbrot set
- `j` for julia set
- `b` for Burning Ship set

and `set` is a number from `0 up to 5` (only used for Julia set)
